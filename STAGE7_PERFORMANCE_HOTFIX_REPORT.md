# STAGE 7 Performance hotfix

## Проблема
Страницы открывались медленно из-за того, что при старте приложения `App.js` статически импортировал все страницы, а через `EstimatePage` / `ZonesPage` / `LibraryPage` подтягивался `src/engine/estimate.js`. Этот модуль импортировал полный `supplierPriceCatalog.js` размером около 49 MB, поэтому даже простая навигация могла ждать парсинг огромного прайс-каталога.

## Что исправлено

1. **Ленивая загрузка страниц**
   - `src/app/App.js` больше не импортирует все страницы на старте.
   - Каждая страница подключается через dynamic `import()` только при переходе на соответствующий маршрут.
   - `src/app/router.js` получил кеширование уже загруженных страниц, защиту от гонок при быстрых переходах и fallback вместо белого экрана.

2. **Полный каталог поставщиков вынесен из критического пути**
   - `src/engine/estimate.js` больше не импортирует `supplierPriceCatalog.js` при обычном запуске.
   - Базовая библиотека и генерация сметы используют быстрый нормализованный каталог `catalog.js`.
   - Полный каталог прайс-листов поставщиков загружается только на странице библиотеки по кнопке `Загрузить прайс-листы`.

3. **Библиотека стала двухрежимной**
   - Быстрый режим: открывается сразу, использует основной каталог.
   - Полный режим: загружает тяжёлый прайс-каталог поставщиков по запросу пользователя.

4. **Проверка устойчивости**
   - `npm test` успешно проходит.
   - Импорт router проверен отдельно.
   - Импорт страницы библиотеки проверен отдельно.
   - Импорт полного прайс-каталога остаётся рабочим и возвращает массив позиций.

## Результат

- Начальный запуск приложения больше не парсит файл прайс-листов поставщиков около 49 MB.
- Страница проектов, паспорт, настройки, КП и проверка открываются без ожидания полного каталога.
- Тяжёлая загрузка осталась только там, где она действительно нужна: в полном режиме библиотеки.

## Как пользоваться

Для быстрого открытия библиотеки используйте страницу `Библиотека` как обычно. Если нужен полный импортированный каталог поставщиков, нажмите `Загрузить прайс-листы` на странице библиотеки.
