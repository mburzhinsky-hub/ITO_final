# Stage 3: UX/UI упрощение пресейл-калькулятора

## Что изменено

- Основной сценарий стал линейным: **Проекты → Паспорт → Зоны → Смета → Проверка → КП**.
- Добавлена верхняя пошаговая навигация `StepNav` со статусами этапов.
- Добавлена компактная сводка проекта `ProjectSummaryBar`: статус, зоны, ориентировочная стоимость, предупреждения.
- Паспорт проекта сокращён до ключевых полей; сложные параметры перенесены в раскрываемый блок «Экспертные параметры».
- Страница зон стала отдельным рабочим экраном с карточками зон, быстрыми действиями и аккуратным блоком типовых зон.
- Страница сметы стала отдельным рабочим местом с KPI, компактным/подробным режимом и сгруппированной таблицей.
- Проверка вынесена в отдельный этап с группировкой предупреждений и действиями.
- Настройки сгруппированы по смысловым разделам: финансы, валюты, ставки работ, коэффициенты, КП, системные параметры.
- Для проектов, зон, сметы, проверки, КП и библиотеки добавлены понятные пустые состояния.
- Из рабочих экранов убраны длинные технические пояснения и служебные формулировки.
- Визуальный стиль сохранён, но интерфейс стал легче: меньше текста, больше явных действий, понятнее статусы.
